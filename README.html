Ответы на вопросы: 1. Куки – строки данных, которые хранятся в браузере.Доступ к куки осуществляется с помощью значения document.cookie. Объекты веб-хранилища localStorage и sessionStorage позволяют хранить намного больше данных, чем куки, так как не отправляются на сервер при каждом запросе. IndexedDB - встроенная база данных, но намного мощнее, чем localStorage. 2.
